## 说清楚：CN2 GT 和 CN2 GIA 的区别

这个很关键，直接影响你的使用体验。

搬瓦工的套餐主要分两个线路档次：

**CN2 GT（Global Transit）**：国际段走 CN2 网络，到了国内骨干网段会回落到普通 163 骨干（AS4134）。早些年还不错，现在晚高峰高峰期间容易出现延迟波动和丢包，但基础稳定性还算可以。这是 $49.99/年 KVM 套餐用的线路。

**CN2 GIA（Global Internet Access）**：全程走 CN2 高端线路，不回落。三网（电信、联通、移动）都有回程优化，晚高峰表现比 GT 稳定不少。这是 CN2 GIA-E 套餐和香港套餐用的线路。

价格差别也摆在这里——CN2 GIA 套餐最低季付 $49.99，年付 $169.99，是入门 KVM 套餐年付价格的三倍多。

一句话总结：**预算有限先用 KVM，对速度和稳定性有要求就上 CN2 GIA-E，香港套餐是土豪选项**。
